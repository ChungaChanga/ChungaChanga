Добро пожаловать! Вы находитесь на презентационной странице букмарклета-переводчика*. Для его использования достаточно добавить следующую ссылку

Переводчик-v.0.1

в закладки браузера и запустить на сторонней странице. На этой странице уже запущен букмарклет (в правом верхнем углу). Вы можете испытать его, попробовав перевести следующий текст: There is no smoke without fire.

Заметьте, что букмарклет ведет себя по-разному при попытках перевести одно отдельное слово и словосочетание.

*Текущая версия букмарклета использует API Яндекс-словаря для перевода отдельных слов, и API Яндекс-переводчика для перевода словосочетаний. Доступ к APIs осуществляется по бесплатным ключам. Исходный код букмарклета.
